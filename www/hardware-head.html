gpsd should work with any GPS using an RS232C or USB interface that advertises NMEA-0183 compliance. Here are some notes on hardware we have tested. Hyperlinks lead to technical information. The "Works with" column is the last gpsd version with which this GPS is known to have been successfully tested; A "*" in this column means we have a regression test load for the device that is checked before each release. Vendors are listed in alphabetical order. There is also a table of receivers which may be appropriate for timing use.

Please mail your success (or failure) reports to gpsd-users using the format described further down this page. Don't forget to include an annotated log so we can include your device in our regression tests.

Warning: the baudrate-hunting code in gpsd tickles serious firmwares bug on some Bluetooth and USB devices, notably those shipped by Holux and including the GPSlim-236. These bugs may send affected GPSes catatonic. See this bug warning for a description of the problem.
