Compatible GPSses

gpsd should work with any GPS using an RS232C or USB interface that advertises NMEA-0183 compliance. Here are the some notes on hardware we have tested:

Name Chipset Interface Sold as gpsd type Tested Sentences Notes
BU-303 SiRF-II USB NavMan, Hyperdata, Deluo n 1.91 GSA, GGL, GSV, RMC, VTG Very generic SiRF II device. Requires pl2303 support which is broken in late 2.4 Linux kernels but good in 2.6.8.
EarthMate Zodiac RS-232 Delorme e 1.10 Unknown Speaks Rockwell binary protocol.
TripMate Zodiac RS-232 Delorme t 1.10 GPRMC, GPGSA, GPGGA, GPGSV, PRWIRID, PRWIZCH Discontinued sometime before November 1998. Takes optional latitude/longitude initialization.
