Q: What chip set does this receiver use?

A: This receiver contains a SiRF chip set. It is programmed to operate in NMEA-0183 upon power up. It can be set to output data in SiRF format. See the documentation on the CD-ROM that is shipped with the receiver.

Q: What NMEA sentences does this unit output in the default mode?

A: $GPGGA, $GPGLL, $GPGSA, $GPRMC, $GPVTG, $GPGSV

Q: Can you tell me if the GPS receiver will output the 1 PPS without connection to RS-232?

A: Yes, 1 PPS is available on Pin 19, Normally low with 100ms TTL high pulse once every second.

Q: Is there a way to disable or remove the RS232 driver?

A: One buyer has reported that you can remove the RS-232 driver chip and there are pads that can be shorted to bring out the TTL serial to connector J1. I would suspect that this is true because these receivers could be ordered with or with out the driver chip, however I have not verified this information.
